# WebApp

The main NeXeZ SiteGeiste web application featuring gamified project management, AI agent interaction, and PixelOffice visualization.

## Tech Stack

- React 18+ with TypeScript
- Next.js 14+ (App Router)
- Zustand for state management
- Tailwind CSS for styling
- Phaser.js for game mechanics

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm --filter webapp dev

# Build for production
pnpm --filter webapp build

# Run tests
pnpm --filter webapp test
```

## Features

- Project management interface
- AI agent interaction
- PixelOffice visualization
- Gamification dashboard
- User onboarding flows
- Achievement system
- Quest management
