# Browser Extension

NeXeZ SiteGeiste browser extension bringing the gamified development experience to your browser.

## Supported Browsers

- Chrome/Chromium
- Firefox
- Edge

## Tech Stack

- Manifest V3
- React for UI
- TypeScript
- Message passing API

## Development

```bash
# Install dependencies
pnpm install

# Start development (Chrome)
pnpm --filter extension dev:chrome

# Build for all browsers
pnpm --filter extension build
```

## Features

- Quick access popup
- Context-aware suggestions
- Developer tools integration
- Cross-browser support
- Offline capability
