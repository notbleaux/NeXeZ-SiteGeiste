# Marketing Website

NeXeZ SiteGeiste marketing and documentation website.

## Tech Stack

- Next.js 14+ (Static Generation)
- MDX for content
- Tailwind CSS
- TypeScript

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm --filter website dev

# Build for production
pnpm --filter website build
```

## Features

- Product information
- Documentation portal
- Blog and updates
- Community showcase
- SEO optimized
