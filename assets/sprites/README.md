# Pixel Sprite Placeholder
This directory will contain pixel art sprites for AI agents and UI elements.

## Sprite Guidelines

- Use 16x16 or 32x32 pixel dimensions
- Maintain consistent color palette
- Create sprite sheets for animations
- Include both light and dark mode variants
- Follow PixelOffice visual style

## Agent Sprites

Each AI agent should have:
- Idle animation
- Working animation
- Thinking animation
- Celebrating animation
- Error state
